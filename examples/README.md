# Kumo framework examples

Complete Vite consumers for the hosted `@cloudflare/kumo-vue`, `@cloudflare/kumo-svelte`, and `@cloudflare/kumo-solid` tarballs.

```sh
cd examples/svelte # or vue / solid
npm install
npm run build
npm run dev
```

These projects install the production HTTPS package artifacts. They do not alias repository source. Select and Tabs are intentionally absent.
