# Minimal explicit hybrid candidate

Axis B architecture candidate with TypeScript held constant. Shared ownership is the existing Select contract/model, tokens/styles and proof vectors. Simple Button scaffolding is generated. Select views remain framework-native and are referenced from `candidates/shared-core`; they are not copied here.

This candidate is deliberately a projection, not a winner. Environment execution is deferred until the merged `shootout/v1` evidence API and repaired Select harness are available.
