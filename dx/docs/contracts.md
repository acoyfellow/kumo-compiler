# Stable DX contracts (v1)
All machine documents carry `schemaVersion: 1`. Paths are project-relative, cannot be absolute, contain `..`, drive prefixes, or NUL. Ownership is deliberately limited to `package`, `generator`, `consumer`, and `native`; this contract does not select an architectural winner. Status vocabulary is `planned`, `not-run`, `passed`, `failed`, `drifted`, `blocked`. Hashes are lowercase `sha256:<hex>` over exact bytes. JSON output is stable, sorted, newline-terminated; diagnostics use stderr. CLI exit codes: 0 success/no drift, 1 drift, 2 usage/config/unsafe path.
