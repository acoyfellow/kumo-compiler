Svelte Button and Field

A complete project that installs the production tarball and renders the same Button and Field contract shown in the live gallery.

Download projectLibrary page

App.svelte

<script>
  import { Button, Field } from '@cloudflare/kumo-svelte';
  import '@cloudflare/kumo-svelte/styles.css';
  let clicks = $state(0);
  let project = $state('Workers');
</script>

<Button onclick={() => clicks += 1}>Clicked {clicks}</Button>
<Button disabled>Disabled</Button>
<Field id="project" label="Project" description="Rendered by Kumo" required bind:value={project} />
<p>Model: {project}</p>

The archive includes a Svelte 5 Vite app and the production tarball URL. For the registry-only result, see Svelte playground validation.

Install

npm install https://kumo-compiler.coey.dev/packages/kumo-svelte-0.0.1.tgz

What this proves

  • Package URL installs through npm
  • Root component exports resolve
  • Package CSS is applied
  • Button and Field run in the target framework

Select is not included. Registry-only playgrounds need public npm publication.
