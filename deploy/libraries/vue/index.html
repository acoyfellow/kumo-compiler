Kumo framework library

Vue component library

Install the packed, independently consumable Vue package and use its native components.

Live package gallery

This gallery is compiled from the installed tarball—not repository source.

Install

npm install https://kumo-compiler.coey.dev/packages/kumo-vue-0.0.1.tgz

Native usage

import { Button, Field } from '@cloudflare/kumo-vue';
import '@cloudflare/kumo-vue/styles.css';

Components

  • Button — native click handling, variants, sizes, loading and disabled states.
  • Field — native model/input handling with label, description, error and ARIA wiring.

Select is not included. Its package eligibility remains under evaluation.
