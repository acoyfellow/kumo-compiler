How-to guide

Install a package

Install the friendly production URL, or pin the content-addressed URL when the exact artifact identity belongs in source control.

Friendly URLs

The friendly URL is stable for the current 0.0.1 package. npm records the resolved URL and integrity in your lockfile.

Immutable URLs

Read the artifact manifest and install its artifact URL:

Content-addressed tarballs are served with immutable caching. The SHA-256 filename equals the artifact bytes.

Import components

Replace svelte with vue or solid. Root and component subpath exports are available:

Verify what was installed

Do not run npm install @cloudflare/kumo-svelte yet. The package is not in the public npm registry; that command cannot resolve it.

CI and private registries

The tarball URL requires no credentials. If your organization mirrors dependencies, mirror the exact content-addressed artifact and preserve its SHA-256. A future registry publication can use the same permanent package name and version only after publication ownership and release policy are approved.
