Styles and tokens

Import one stylesheet for complete default presentation. Import tokens separately only when you need direct access to the custom properties.

Default stylesheet

The Vue and Svelte stylesheets include fallbacks or their token stylesheet. Solid’s stylesheet imports tokens. All three produce visible Button states and Field borders without extra setup.

Token stylesheet

Token familyPurpose
--kumo-color-*Primary, danger, text, muted, and border colors.
--kumo-space-*Internal spacing.
--kumo-radiusControl corner radius.

Override tokens

Override tokens after the package stylesheet. Do not target framework-generated internals. Stable class names use the kumo- prefix, but tokens are the intended theme seam.

Asset behavior

The package contains CSS only—no fonts, images, or remote runtime assets. Packed consumer proofs verify CSS is emitted once and all runtime assets return successfully.
