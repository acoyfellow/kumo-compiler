Distribution

Friendly tarballs use five-minute caching. Content-addressed tarballs use immutable one-year caching. All package responses use application/gzip and attachment disposition. The manifest is available at /packages/manifest.json.

Current non-goals

  • No public npm registry publication yet.
  • No compatibility promise for components outside Button and Field.
  • No claim that the package version itself expresses readiness.
