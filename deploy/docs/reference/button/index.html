Button

A framework-native button with Kumo variants, sizes, loading text, disabled behavior, and forwarded native attributes.

Props

PropTypeDefaultBehavior
variantprimary | secondary | dangerprimaryVisual intent.
sizesmall | medium | largemediumPadding and text scale.
disabledbooleanfalseNative disabled state; handlers do not fire.
loadingbooleanfalseDisables the control and announces loading text.
typebutton | submit | resetbuttonNative button type.

Events

Use the framework’s native event syntax: onclick in Svelte, @click in Vue, and onClick in Solid.

Accessibility

  • Renders a native <button>.
  • Has a minimum 44px control height.
  • Preserves native keyboard activation and disabled semantics.
  • Requires a meaningful accessible name from its child content.
