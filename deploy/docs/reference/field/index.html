Field

A labeled native input with description, error, required, disabled, model, and ARIA wiring.

Props

PropTypeRequiredBehavior
idstringyesInput ID and base for description/error IDs.
labelrenderable textyesVisible native label.
descriptionrenderable textnoCreates <id>-description.
errorrenderable textnoSets invalid state and creates an alert.
requiredbooleannoNative required state and visible marker.
disabledbooleannoNative disabled state.
value / modelValuestring | numbernoFramework-native model value.

ARIA contract

Description and error IDs are joined in aria-describedby. An error sets aria-invalid="true" and renders with role="alert". The label’s for always targets the input ID.

Native attributes

Input type, name, autocomplete, placeholder, inputmode, and other supported native input attributes are forwarded. Field does not create form validation policy.

Framework models

  • Svelte: bind:value
  • Vue: v-model
  • Solid: controlled value and onInput
