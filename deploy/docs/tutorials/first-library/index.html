3 — Check the contract

  • The button starts at Clicked 0 and increments.
  • The label targets the input ID.
  • Description and error IDs are composed into aria-describedby.
  • required, disabled, and aria-invalid reach native controls.
  • The package stylesheet supplies visible focus, state, spacing, and minimum 44px control height.
