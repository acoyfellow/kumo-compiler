Current decision

Keep TypeScript as the reference compiler during the architectural bake-off. It is the only full emitter and browser-backed product baseline today. Go and Zig remain serious future core candidates; planner-only timing is not enough to justify a rewrite.

Read the decision record →

Methodology

Every JSON record contains 2 discarded warmups and 10 measured samples. Each sample uses a fresh Node process and fresh output root; the first measured sample is labeled cold, and wall time includes process startup. Runs were recorded on Apple M4 Pro, arm64, macOS 25.4.0 with Node v24.17.0.

All candidates report 41 components, 164 target plans, and 164 deterministic output files. Output byte totals and hashes differ because artifact formats and workload scope differ. Portable startup, CPU, and peak RSS telemetry were not captured; zero CPU fields and null RSS fields are displayed as unavailable rather than as measured zero.
