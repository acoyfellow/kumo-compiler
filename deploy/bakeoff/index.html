Interpretation limits

  • The internal TypeScript baseline is an existing authority-backed control, not a new run in this pilot.
  • Mitosis generated 28 sources, but only Button’s four framework targets have build and browser interaction proof.
  • Shared-core’s 116 passed gates cover reducers/contracts, types, and maintainability. They never imply a browser pass.
  • No candidate should receive credit for a not-run gate, and no “partial” cell is equivalent to passed.
